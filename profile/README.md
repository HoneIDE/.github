# Hone IDE

A native, AI-powered code editor. Fast. Extensible. Built for real workflows.

## Repositories

| Repo | Purpose |
|------|---------|
| [extensions](https://github.com/HoneIDE/extensions) | Built-in language extensions — TypeScript, Python, Rust, Go, C++, and more |
| [themes](https://github.com/HoneIDE/themes) | VSCode-compatible color themes with import tooling |
| [api](https://github.com/HoneIDE/api) | Public extension API — zero runtime, pure TypeScript types |
| [landing](https://github.com/HoneIDE/landing) | Marketing and landing page |

## Contributing

- Read the [Extension API docs](https://github.com/HoneIDE/api)
- Browse [built-in extensions](https://github.com/HoneIDE/extensions) to see patterns
- Submit a theme via the [themes repo](https://github.com/HoneIDE/themes)
